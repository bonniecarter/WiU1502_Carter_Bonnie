Project Name: WorldSurf

By Bonnie Carter

Project Description

This project is a design project for Full Sail University in regards to usability testing. We will be utilizing the skills learned at Full Sail (html, css & java scripting), Navflow to test our projects & GitHub for sharing content.

How do you feel about the quality of content, quantity of content, usability, can you even figure out why you are there or what you should be "doing" while at a site you have visited. Have you ever asked yourself, "did anyone test this site before launching it?" Don't let these questions be asked about your site. Always test!
